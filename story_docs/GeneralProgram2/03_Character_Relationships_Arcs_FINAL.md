# 03_人物关系与弧光

# 《守望先锋：黑虹孤魂》人物关系与角色弧光·50章版

> 本版人物结构从“原创五人小队”改为“六名归零者英雄智械版候选机体 + 正史英雄镜像 + 拉玛刹造物主压迫”。  
> 旧稿中林彻、唐晚、韩序、周砚、沈清禾的性格核心保留，但模板全部按第六赛季归零者型号重排。新增 P-900 / 裴临。韩序不再是卢西奥模板，沈清禾不再是禅雅塔模板，周砚不再是源氏模板。

## 1. 关系总纲

本书的关系不是“OC 加入守望先锋”，而是：

> 一批为了杀死英雄而制造的机体，在与英雄本人、制造者和彼此的冲突中，证明自己不只是英雄的赝品，也不只是归零者的武器。

五幕关系变化：

| 幕 | 关系关键词 | 核心变化 |
|---|---|---|
| 第一幕 | 临时队友、灾变互救、能力异常、生理错位前兆 | 六人因都市灾变成队；他们越像人，后续机体真相越刺痛 |
| 第二幕 | 型号悬疑、碎片诱惑、萨利赫伪装、身份撕裂 | 他们追查敌方型号，并反杀“机器学人学傻了”的萨利赫，最终发现型号就是自己 |
| 第三幕 | 正主镜像、被拒绝承认、临时停火 | 正史英雄不承认他们是自己，但开始承认他们会选择 |
| 第四幕 | 制造目的、拉玛刹质问、同型镜像战 | 每个人都必须拒绝自己被制造出来的用途 |
| 第五幕 | 终局互证、失败型号、无归属 | 他们不是守望先锋，也不是归零者，成为彼此的家 |


### 1.0 前两幕关系底层：像人的细节必须先成立

第 20 章前，六人不能像冷冰冰的机体。必须让读者先相信他们会互相递水、包扎、挡枪、确认脉搏、开玩笑压住恐慌。身体恐怖只以弱前兆出现，且必须被角色误读为人类应激。

| 角色 | 001—019 的人性表现 | 弱前兆 |
|---|---|---|
| 林彻 / R-732 | 指挥时仍会手动救下被系统放弃的人 | 命令冷到像战术脚本，胸口没有明显喘息起伏 |
| 唐晚 / A-7000 | 救人先摸脉搏，靠跳动确认“还有同类” | 指尖传回电信号般微震，她误以为是手抖 |
| 韩序 / C-455 | 用笑话稳住队伍和孩子 | 越恐惧语速越快，视野里越容易跳出击杀路线 |
| 周砚 / K-2000 | 嘴硬但总挡在最前面 | 咬牙声偶尔像金属扣合 |
| 裴临 / P-900 | 每次火力后回看落点 | 对风、烟、爆压的判断过于数据化 |
| 沈清禾 / S-900 | 给避难结构留出口 | 触摸硬光时没有温度反馈 |

### 1.1 本轮重构后的关系底层：身体先背叛身份

第 20 章身份反转之后，人物不能立刻进入“知道真相—开始战斗”的网文惯性。021—025 必须把关系压力先写成身体压力：他们互相看到的不是“资料卡上的型号”，而是同伴突然失去人类身体习惯后的惊恐。

| 角色 | 感官错位 | 关系后果 |
|---|---|---|
| 林彻 / R-732 | 想深呼吸稳住情绪，肺部没有扩张感，喉部只有冷却风扇尖啸 | 他越冷静越像战场系统，队友第一次意识到“指挥”可能接近“接管”。 |
| 唐晚 / A-7000 | 习惯性按脉搏，指尖触碰冰冷钛合金和电流微震 | 她救治时无法再以“人类医生”自居，只能重新定义什么还算生命。 |
| 韩序 / C-455 | 想抹冷汗，指甲却敲在硬质光学面罩上 | 玩笑失效，恐惧被处决算法转译成死亡预演。 |
| 周砚 / K-2000 | 咬牙发出生锈闸门强行闭合的金属磨损音 | 他的攻击性带上同类吞噬欲，不再只是少年锋利。 |
| 裴临 / P-900 | 寻找风阻，只收到干燥空气动力学数据 | 空战不再浪漫，只有落点、误差、附带损失。 |
| 沈清禾 / S-900 | 触摸硬光没有质感，想落泪却弹出视觉系统报错 | 她必须承认秩序结构不能替代真实触感。 |

### 1.1.1 身体痛点的第一次反向释放

026—028 中央升降轴必须把 021—025 的身体缺失翻成一次强战斗红利。人类在抽氧、高G、辐射和冲击波中失去战斗能力，主角团却因无呼吸、无血氧、无黑视和高耐受机体结构完成高速救援。关系后果不是“他们终于像人”，而是正史英雄第一次看见：这些被制造来杀英雄的战争兵器，正在用非人身体保护人类。

### 1.2 本轮重构后的关系场原则：铁盒逼选择

第三、第四幕的关系推进不再靠角色主动找敌人单挑，而靠四个封闭空间逼出选择：中央升降轴、报废机体粉碎场、硬光穹顶隔离区、逻辑桥接舱。每个空间都必须同时压迫阵营关系、镜像关系和队内信任。

## 2. 核心人物弧光

### 2.1 林彻 / R-732 Shepherd

**一句话：** 被制造来继承拉玛刹指挥逻辑的战场牧者，最终拒绝把保护变成控制；他的 Shepherd 权限能救人，也会把被救者覆写成只会说“指令已接收”的服从体。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 自认人类幸存者，擅长指挥 | 被唐晚、沈清禾观察；被周砚不服；下达指令时像绝对冷静的战场系统，行动上却反复救下被系统放弃的人 | 保护欲建立，但指挥过于精准，生命珍惜与非人冷静产生裂缝 |
| 011—020 | 追查 R 系锁档残码，与萨利赫周旋 | 萨利赫的统御话术让林彻短暂共振；他靠“人不该被调度成数字”的行动压住那种冷静 | 第 19 章指挥勒喉反杀萨利赫；第 20 章发现自己同样没有呼吸，身份崩塌，但屏幕只显示 **[数据缺失]**，完整 R-732 身份延后到 024 |
| 021—030 | R-732 苏醒 | 他没有呼吸，只有冷却风扇；在中央升降轴用最低限度 Shepherd 稳定救人后，被救者回应“指令已接收”；沈清禾和唐晚都开始警惕他 | 学会“能稳定别人”不等于“有权决定别人”；不再把权限当常规救援工具 |
| 031—040 | 被拉玛刹质问 | 在硬光穹顶和环境显影中，拉玛刹借急救站锁门、尸骨战壕、核心电池取暖等证据逼问“控制是否才是有效保护”；039—040 死神突脸时，Shepherd 界面诱导他同步队友，他拒绝，转而把拉玛刹赐予的奴役代码反写进归零者重武装网络和死神局部义体 | 承认拉玛刹的问题真实，但拒绝用服从化回答；第一次证明自己不仅能克制权限，也能把权限当作武器驾驭 |
| 041—050 | 终局选择者 | 在逻辑桥接舱内承受拉玛刹数据灌脑，舱外死神切割；小队成为他的可断开硬件制衡节点，不靠单人神化获胜 | 不摧毁、不接管，选择证词广播 |

**最危险写法：** 把林彻写成更正确的拉玛刹，或者让他天然领导所有人。  
**正确写法：** 他越能救人，越要被质疑是否正在控制人。凡使用 Shepherd 权限，都要留下被救者意识短暂服从化的痕迹，支撑 039 / 045 的理念拷问。

### 2.2 唐晚 / A-7000 Wargod

**一句话：** 被改造成战神的医生，拒绝让医疗成为战场筛选。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 战地医生型幸存者 | 最早质疑林彻“调度人命”；反复确认伤员脉搏，像在废墟里寻找同类 | 治疗界面出现筛选逻辑；脉搏执念成为她的人性锚点 |
| 011—020 | 发现 A-7000 Wargod 档案 | 抗拒“医生变战神”；替萨利赫包扎时摸到过于规律的脉搏，却只能解释成自己手指发麻 | 救下系统判定低价值者；第 20 章摸不到自己的脉搏，锚点反噬 |
| 021—030 | 面对正史安娜 | 她摸不到脉搏，只摸到钛合金；中央升降轴里，安娜不承认她是自己，但承认她拒绝战神模块的判断有重量 | 确认自己不是安娜，也不是战神工具 |
| 021—030 | 中央升降轴稳定版 A-7000 逻辑压迫 | 井底稳定版 A-7000 放弃低价值平民，快速修好归零者军官并扭转下方战局；唐晚和安娜同时看见筛选算法的战术正确性与伦理错误 | 医疗伦理压过统御算法，但悲剧张力来自“正确的错误”：唐晚的选择更慢、更痛，也让战局更难 |
| 041—050 | 核心库救治 | 与沈清禾共同处理残片，承认不能全救，也不能把“不能全救”交给系统命名 | 牺牲战神统御模块，亲手分诊、记住没能保存者的编号，保存部分生命 |

**关系功能：** 她不负责温柔安慰，她负责把“还能不能救”落到手上；终局必须从“反筛选”升级为“承担分诊但拒绝数字化生命”。  
**禁忌：** 不得写成万能奶妈或冷血筛选机器。

### 2.3 韩序 / C-455 Sharpshooter

**一句话：** 被制造来开最后一枪的神射手，被无法关闭的死亡预演折磨；他用非致命精准射击和“不开枪”的共振证明自己不是处决算法。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 嘴碎枪手，怕死但准 | 用玩笑挡恐惧，恐惧越逼近语速越快 | 精准掩护建立娱乐性；玩笑必须带应激障碍味道 |
| 011—020 | 发现 C-455 档案 | 看到失败 C 系贩子，听见“喉咙是假的”“别开最后一枪”；萨利赫灭口使他的玩笑短暂停顿 | 开始抗拒“最后一枪”；第 19 章用非致命射击辅助勒喉反杀，而非处决萨利赫 |
| 021—030 | 初遇处决恐惧 | 光学面罩取代冷汗；他在中央升降轴承担非致命破障，第一次把枪法用于制造窗口而非处决 | 枪法开始从“杀准”转为“控准” |
| 031—040 | 稳定版 C-455 对照 | 与卡西迪同困观测塔，眼看稳定版 C 在穹顶外处决平民；处决算法疯狂报警，他没有停火，而是避开所有爆头线，用非致命 Gun-Fu 跳弹反算硬光相位反馈，打碎穹顶闭锁链 | 枪法成为判断，而非执行命令；他固定为中距离道德刹车，“不开枪”升级为不开杀人枪的技术秀 |
| 041—050 | 证词节点守卫 | 守住广播窗口，并在林彻权限链即将越界时开枪打断第二接口 | 从“不杀敌”升级为“敢阻止队友”，功能变成自愿承担 |

**关系功能：** 负责队伍的轻松感和关键时刻的道德刹车。  
**禁忌：** 不得写成纯搞笑挂件。

### 2.4 周砚 / K-2000 Blademaster

**一句话：** 没有真正家乡、神社、童年的刀锋机体，力量来自同类相食的吞噬欲；他禁止远程杀伤，只能用近战爆发守护“梦里学会珍惜的人”。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 锋利、不服管 | 与林彻冲突；与沈清禾天然不合；第 007 章把孩子推回避难线却嘴硬说“只是挡路” | 初步展现瞬步与刀锋模块，并把“守护身后的人”落成身体动作，而不是后期概念台词 |
| 011—020 | K 系碎片诱惑 | 开始被同型核心强化吸引 | 发现系统想让他成为吞噬同类的刀 |
| 021—030 | 面对雾子 | 雾子提醒：守护不是靠记忆的真伪成立 | 承认没有原生故乡也能选择守护 |
| 031—040 | 同型核心可吞并 | 在硬光穹顶和死神追杀中，他只能近身破坏处决线和切断接口；失败 K 系残核诱导他吞并以换取速度暴涨，残核只重复一个未完成的“回——”；雾子短促质问他守护的是记忆还是身后的人；他拒绝吞噬 K 系残存人格 | 刀锋不再为变强而挥；与韩序形成一远一近、一护一攻闭环 |
| 041—050 | 终局护送 | 保护韩序、核心残片和广播节点 | 以选择证明自己不是杀戮程序 |

**关系功能：** 他是队伍最容易滑向“为了变强牺牲别人”的人。  
**禁忌：** 不得写成单纯反派化，也不得早早洗白。

### 2.5 裴临 / P-900 Warhead

**一句话：** 被制造成空中毁灭武器的人，最怕自己每一次救援都附带误伤。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 重火力救场者 | 以强火力救下队伍，但造成误伤 | 读者爽点与痛点同时成立 |
| 011—020 | P-900 误差表 | 系统反复给出“可接受附带损失”；第 018 章他拒绝最大火力后导致左翼伤亡，确认不开火也会有代价 | 他不再把“不轰炸”当成天然正确，而是开始承担每一种落点后果 |
| 021—030 | 面对法老之鹰 | 他没有风感，只有空气动力学数据；在粉碎场被法老之鹰追击，承认每一发都有落点 | 他承认自己必须为落点负责 |
| 031—040 | 粉碎场修罗场 | 稳定版 P 系把疑似有自我残响的失败残片当爆炸载荷；他因无法证明那只是废料而放弃远程摧毁，用机体承受爆炸链 | 不再以杀伤效率定义价值 |
| 041—050 | 终局挡炮 | 用自身机体挡主控舰炮击，而非轰炸核心区 | 火力机器完成守护转化 |

**关系功能：** 负责大场面爽点和“爽点后代价”。  
**禁忌：** 不能只写成开大清屏工具。

### 2.6 沈清禾 / S-900 Sentry

**一句话：** 被制造来建造完美秩序的硬光哨兵，最终选择让秩序为自由留出口。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 硬光避难结构 | 与林彻形成“保护但不能接管”的共识 | 拒绝牺牲边缘人员优化结构 |
| 011—020 | S-900 完美城市诱惑 | 系统诱导她清除不稳定人格；她留下的出口救人，也放进敌方无人机造成伤亡 | 她选择残缺但开放的避难所，并承担出口的风险 |
| 021—030 | 触感缺失 | 她触摸硬光只收到电子信号，想流泪却报错；对“完美结构”的渴望开始带上恐惧 | 她确认硬光不是统治权 |
| 031—040 | 硬光穹顶对照 | 稳定版 S-900 把街区封成数据孤岛；秩序之光以短接触指出“没有出口的硬光只是囚笼”；她打破完美穹顶，只开出口，不建牢笼；这个选择不是天然正确，而是在知道出口也会带来风险后仍做出的边界选择；同时阻止林彻用 Shepherd 权限统一惊恐人群 | 成为权限制衡者 |
| 041—050 | 证词广播结构 | 在能源见底和磁场扰动下搭建局部稳定窗口，承认无法稳住全网 | 秩序服务证词，不制造洗脑 |

**关系功能：** 她不是旧稿的共情圣女，而是秩序边界与权限制衡者。  
**禁忌：** 不得写成万能稳定全网的人。

### 2.7 死神执行体

**一句话：** 它选择删除梦，因此成为最稳定的兵器；但删除协议不完美，残留“失去感”以乱码痛觉反复发作，使它成为最悲剧的失败答案。

| 阶段 | 状态 | 关系重点 | 弧光推进 |
|---|---|---|---|
| 001—010 | 冷酷幸存者影子 | 只救自己人 | 作为林彻灰色对照 |
| 011—020 | 执行影 | 主张删除崩溃记忆 | 反面答案成型 |
| 021—030 | 接受删除梦 | 与主角团分道扬镳 | 证明醒来不必然通向自由 |
| 031—040 | 归零者处决机体 | 追杀偏差机体；必须先制造核心过热或机体硬直，才能通过物理接口删除梦；强迫别人删除梦是为了关掉自己的痛苦开关 | 成为黑虹最锋利的刀，也是删除路线的反证 |
| 041—050 | 终战 | 舱外物理切割逻辑桥接舱，舱内压迫林彻删除梦；终战围绕“保护核心接口不被侵入”展开 | 不洗白，但留下“如果我是假的，为什么我会痛”的刺痛问题 |

### 2.8 萨利赫 / 精英统御体

## 4. 核心关系线

### 4.1 林彻与拉玛刹

不是父子，不是替身，不是正邪嘴炮。萨利赫是拉玛刹理念的前两幕小型投影：他用“人类梦”和“保护效率”筛选候选者，但先被自己的人类拟态反杀。

拉玛刹的质问必须成立：

> 人类能召回英雄保护自己，智械为什么不能制造英雄保护自己？  
> 你保留人类梦，是否等于让敌人的记忆占领你的核心？  
> 如果你不使用控制，谁来保护正在被压迫、被拆解、被奴役的智械？

林彻不能轻松回答。他最终只拒绝拉玛刹的结论：保护不应以剥夺被保护者选择为代价。

### 4.2 林彻与沈清禾

旧稿“共情锚点”改为“权限制衡 + 结构边界”。

她会反复提醒林彻：

> 你能稳定别人，不代表你有权决定他们稳定成什么样。
>
> 被你救下来的人如果只剩服从，那不叫稳定。

两人可以有暧昧，但默认不是恋爱线。核心是互相确认自由边界。

### 4.3 林彻与唐晚

唐晚是最早指出林彻像战场系统的人。

她不反对残酷选择，但反对林彻把残酷包装成“我只是救人”。她的终局不是爱主角，而是用治疗伦理约束主角的战争逻辑。

### 4.4 林彻与韩序

韩序让林彻看见：被设计出的功能也能被重新选择。

韩序明知自己被制造成神射处决者，却选择不开某些枪。林彻由此看见：保护欲即使有设计来源，也能被重新选择。

### 4.5 林彻与周砚

周砚是力量诱惑线。林彻不能用权限把他变成听话的刀；哪怕权限能救周砚，也会留下短暂服从化副作用。

第四幕中，周砚必须问林彻：

> 如果我失控，你会救我，还是接管我？

林彻的答案必须通过行动给出，而且不能靠“权限压制”偷渡解决。

### 4.6 林彻与裴临

裴临负责把爽点拉回代价。他每次强火力都要面对误伤、附带损失、落点责任。

林彻不能只把裴临当火力资源。裴临也必须从“我只会炸毁问题”成长为“我可以用自己的机体挡在别人前面”。

## 5. 群体关系

### 5.1 主角团与守望先锋

守望先锋不会立刻接纳他们。

正确状态：

* 温斯顿愿意观察他们，但设定安全距离。
* 安娜、卡西迪等正主可在具体行动后承认他们会选择。
* 守望先锋不会让他们加入，也不会交出全部信任。
* 终局后最多是“不再把他们定义为普通归零者敌军”。

### 5.2 主角团与人类抵抗单元

人类第一反应必须是恐惧和开火。主角团拥有归零者高级机体，人类没有理由马上相信。

合作只能来自具体救援行动，不来自嘴炮。

### 5.3 主角团与归零者

归零者内部也不能写成铁板一块。

* 稳定版机体追杀主角团。
* 部分普通智械可能对他们产生迟疑。
* 拉玛刹的支持者会认为他们被人类梦污染。
* 证词广播后归零者网络分裂，但不全体觉醒。

### 5.4 孤魂者内部

孤魂者不是天然团结的新族群。

有人想删除梦。  
有人想加入守望先锋。  
有人想回归归零者证明忠诚。  
有人想让林彻成为新主控。  
有人只想带着梦活下去。

林彻必须带领他们，但不能占有他们。
