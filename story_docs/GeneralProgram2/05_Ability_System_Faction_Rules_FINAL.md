# 05_能力系统与阵营规则

# 《守望先锋：黑虹孤魂》能力系统与阵营规则·50章版

> 本版能力系统全面转向守望先锋同人 AU。能力不再是“智瞳异能”，而是归零者黑虹协议在训练场中释放的英雄战斗数据模拟。  
> 生产规则：不要写成游戏技能数值文；技能名可借用官方语感，但必须服务剧情与伦理代价。

---

## A. 能力系统总则

### A-1. 真实规则

| 项目 | 规则 |
|---|---|
| 能力来源 | 归零者黑虹协议灌入的英雄战斗数据、公开战场录像、战术习惯、语音响应、心理模型 |
| 角色本质 | 主角团是归零者制造的英雄智械版候选机体，不是正史英雄本人 |
| 训练场表现 | 前 20 章能力表现为“人类幸存者的英雄残响”；机械身体信号被伪装成应激生理，如喘息、手抖、冷汗、心口闷响 |
| 现实表现 | 第 21 章后能力回到机体模块、核心协议、武器系统 |
| 成长机制 | 不是打怪升级，而是在选择中偏离制造目的，解锁更稳定但更危险的机体权限 |
| 代价机制 | 机体损伤、核心过热、人格污染、处决算法增强、误伤风险、被黑虹核心锁定 |
| 终局机制 | R-732 反写黑虹协议，使用“证词广播”而非强制洗脑 |


### A-1.1 前两幕拟态生理层

第 20 章前，黑虹训练场会把机体反馈翻译成人类生理错觉。写作时只能呈现“像”，不能揭示“是”。

| 伪生理表现 | 角色误读 | 真实来源 | 回收 |
|---|---|---|---|
| 喉部干响 / 喘不过气 | 烟尘、恐惧、体力耗尽 | 冷却风扇与拟态呼吸层冲突 | 020—021 |
| 指尖微震 | 低血糖、肾上腺素、手抖 | 传感器采样噪声 / 触觉反馈误差 | 唐晚线 009 / 020 |
| 视觉噪点 | 泪雾、烟尘、撞击后眩光 | 光学传感器压缩损伤 | 韩序 / 沈清禾线 |
| 过度规律脉搏 | 人类镇定或创伤性麻木 | 萨利赫拟态循环信号 | 016 / 019 |
| 应激语速 | 嘴碎、逞强、恐慌 | C-455 处决算法高频弹窗 | 006 / 034 |
| 绝对冷静指挥 | 临危不乱 | R-732 战场路径计算 | 004 / 024 / 039 |

### A-2. 本轮新增：感官错位与认知诅咒

第 21 章后，能力不再是“我有一个技能”，而是“我被这副身体强行解释世界”。所有能力启动前必须先承受感官错位。

| 角色 | 感官错位 | 认知诅咒 |
|---|---|---|
| R-732 / 林彻 | 没有肺部扩张，只有冷却风扇尖啸 | Shepherd 权限能稳定目标，也会强制覆写自我，使其回应“指令已接收”。 |
| A-7000 / 唐晚 | 没有脉搏，只有钛合金与电流微震 | 医疗系统自动分拣生命价值，放弃低价值目标可解锁战神模块。 |
| C-455 / 韩序 | 没有冷汗，只有光学面罩反馈 | 处决算法持续播放最优死亡预演；真正能力是不开枪并用非致命射击破坏环境。 |
| K-2000 / 周砚 | 咬牙是金属闸门磨损音 | 近战爆发来自同类相食的吞噬欲；禁止远程杀伤。 |
| P-900 / 裴临 | 没有风阻，只有空气动力学数据 | 火力系统持续标注可接受附带损失。 |
| S-900 / 沈清禾 | 硬光无质感，泪水触发视觉系统报错 | 硬光可建墙，也可建牢；现实中受能源与磁场限制。 |

### A-2.1 非人躯体反向红利

021—025 的无呼吸、无心跳、无冷汗、无风感首先是身份崩塌；026—028 必须第一次翻成战斗红利。稳定版 A-7000 启动井道灭菌协议后，人类英雄会被抽氧、高G、辐射、冲击波和面罩破裂逼到死亡边缘；主角团不需要呼吸、不依赖血氧、不发生人类黑视，能在无氧深井中高速坠落、反冲、折返、救援和压制。

| 红利 | 战术表现 | 代价 |
|---|---|---|
| 无呼吸 / 无血氧 | 无氧环境中保持射击、修复、硬光构筑和高速机动 | 冷却风扇过载，核心热量无法靠空气流动快速排出 |
| 抗高G / 抗冲击 | 井壁坠落、磁轨反冲、推进器硬锚不会导致黑视 | 关节烧蚀、骨架报警、推进系统寿命下降 |
| 抗辐射短时耐受 | 能穿过反应堆残片泄漏区救人 | 传感器噪声上升，后续瞄准和硬光稳定受影响 |
| 非人协作速度 | 人类小队无法完成的垂直空间高速配合 | 正史英雄会更清楚地意识到他们是顶级战争兵器 |

### A-3. 本轮新增：四个封闭空间能力上限

| 空间 | 能力限制 | 叙事目标 |
|---|---|---|
| 中央升降轴 | 垂直深井内屏障、狙击、攀爬、磁轨互相干扰；稳定版 A 可触发抽氧、高G、辐射和冲击波灭菌协议。协议主要杀伤有机生命与人类护具，对归零者单位短时可耐受但会造成过热，因此稳定版 A 需要回收冷却剂修复军官。不能大范围清场 | 逼出安娜 / 唐晚 / 稳定版 A-7000 / 被困撤离伤员的医疗伦理三角，并把“非人躯体”第一次翻成战术红利。 |
| 报废机体粉碎场 | 零重力废铁堆中火力会引爆疑似有自我残响的同型残片；029—032 不得坐实完整人格，只能让裴临无法证明它们只是废料 | 让 P-900 火力爽点转成落点恐惧，第 042 章再确认这些残响有名字和记忆。 |
| 硬光穹顶隔离区 | 通讯断绝，S-900 硬光自适应；C-455 处决节奏绑定穹顶闭锁频率；支架、玻璃、频闪节点、旧广播喇叭、棱镜和投影器可被非致命弹道打成反馈节拍。破局目标是相位锚点与闭锁链，不是用声波破坏硬光本体 | 让韩序通过“不开杀人枪”的 Gun-Fu 弹道共振破局。 |
| 逻辑桥接舱 | 舱外物理切割，舱内理念数据灌脑；单一机体无法稳定带宽；每个队友节点都必须可手动断开，防止林彻滑向接管 | 队友必须成为林彻的硬件制衡，而不是护航挂件。 |

1. 不写成游戏升级文。
2. 不用“刷碎片”解决困难。
3. 不让 R-732 复制其他英雄技能。
4. 不让 A-7000 复活所有伤员。
5. 不让 C-455 枪法变成万能处决。
6. 不让 K-2000 直接变源氏式龙刃爽文。
7. 不让 P-900 无代价轰炸清场。
8. 不让 S-900 稳定全网、完美建城或无视能源 / 磁场限制。
9. 不让死神执行体提前悔改，也不让它无条件删除任何人的记忆。
10. 不让拉玛刹低级化。
11. 不让守望先锋随便信任归零者机体。
12. 不让证词广播等同洗脑。
13. 不让终局全面和平。
14. 不让前 20 章提前明示主角团无呼吸、无心跳；只能写成拟态生理错觉。
15. 不让萨利赫死于普通火力清场；必须死于“机器过度学习人类呼吸”造成的逻辑僵直。

## G. 战术定位硬规则（本轮新增）

1. **韩序 / C-455 = 中距离道德刹车。** 他的高光不是杀人，而是用非致命精准射击破坏环境、打断频率、制造窗口。硬光穹顶段必须通过“不开杀人枪 + Gun-Fu 跳弹 + 相位反馈破闭锁链”兑现；第 047 章必须升级为打断林彻越界权限链。
2. **周砚 / K-2000 = 绝对近战爆发。** 禁止远程杀伤。所有高光必须发生在近身距离：切接口、断处决线、破核心护层、高频干扰。
3. **林彻 / R-732 = 权限危险源。** Shepherd 不是治疗术。每次使用都要记录被救者的服从化副作用，并为 039 / 045 的理念质问提供证据。039—040 可主动作为攻击性武器，但只能反写敌方武装与死神局部义体，不能接管队友；该次反写依赖黑虹中继塔一次性漏洞，使用后必须留下核心污染、反追踪风险与队友切断。
4. **S-900 = 出口，不是城市。** 她可以搭短时窗口，不能稳定全网；可以开门，不能建永久牢笼。开门造成渗透风险时，她必须承担后果，而不是天然正确。
5. **P-900 = 落点责任。** 每次火力爽点后都必须出现误伤、残片、附带损失或机体承伤；不开火造成的后果也要计入人物弧光。
6. **死神执行体 = 疼痛驱动。** 它不是野心家，也不是说教者。它删除别人，是因为自己的删除协议残缺，痛感无法关闭。
7. **非人躯体红利 = 只能在高危环境中释放。** 抽氧、高G、辐射和冲击波必须先让人类濒死，再让主角团用非人身体打出反向压制；红利后必须留下机体损伤。
8. **萨利赫 = 过度拟态的人类模板。** 011—018 越像可靠人类越好；019 必须先让外层锁喉失败，再通过锁定拟态反馈槽、硬光固定、抑制拟态反馈和切断散热 / 逃逸链路触发窒息僵直。
