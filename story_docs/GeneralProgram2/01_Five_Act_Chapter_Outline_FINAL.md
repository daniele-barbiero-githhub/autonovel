# 第一幕：召回令后的都市（001—010）

## 幕级执行约束

| 项目 | 内容 |
|---|---|
| 读者表层认知 | 这是守望先锋第六赛季“智械入侵” AU；归零者全球入侵正在扩散；临海市像真实城市 |
| 角色认知 | 自己是人类幸存者，能力是某种“英雄残响 / 战场应激”；身体异常只能被理解成灾后应激、脱水、低血糖或恐慌反应 |
| 前两幕风格硬规 | 正文禁止直接写“悲伤、害怕、绝望”等抽象情绪。必须用机体反应的伪装版表现：心口散热般的闷响、光学噪点误认为泪雾、指尖微震误认为低血糖、喉部风扇声误认为喘息。第 20 章前不得揭示其机械本质。 |
| 对比张力 | 第 20 章前要把六人的人性写得更温暖、更脆弱：递水、包扎、互相挡火、确认脉搏、开玩笑压住恐慌。越像人，020 的编号揭露越痛。 |
| 必须出现 | 召回令新闻、归零者、里约 / 多伦多 / 哥德堡数据影子、六名主角能力雏形；001—003 章要连续出现守望先锋视觉残留，如温斯顿召回令坏屏重播、裂空弧光残影、旧守望频道串台 |
| 必须埋下 | 城市地理拼接异常、型号编号残影、黑色虹膜 UI、R 系保护指令残码、电子哀鸣、唐晚对脉搏的异常执着、韩序恐惧时语速过快、林彻下达指令时不属于人类的绝对冷静；完整 R-732 Shepherd 不得在 024 前坐实 |
| 禁止 | 明说黑虹协议、主角是机体、A/C/K/P/S/R 是自己、正史英雄出场对峙；禁止让角色主动怀疑“自己不是人” |
| 幕末硬结果 | P-900 以重火力救场但造成误伤；众人意识到“英雄能力”也可能是敌方武器 |

| 角色认知 | 以为自己在调查归零者制造的敌方精英机体；以为萨利赫是掌握城市避难网的旧市政协调员 / 情报贩子 / 幸存者领袖 |
